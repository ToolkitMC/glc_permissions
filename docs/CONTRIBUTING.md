# 🤝 Katkıda Bulunma Rehberi

GULCE Admin Power projesine katkıda bulunmak istediğiniz için teşekkürler!

## 🐛 Hata Bildirme

1. [Issue](https://github.com/ToolkitMC/glc_permissions/issues) açın
2. Hatayı detaylı açıklayın
3. Minecraft versiyonunu belirtin
4. Hata loglarını ekleyin

## ✨ Özellik Önerisi

1. [Issue](https://github.com/ToolkitMC/glc_permissions/issues) açın
2. "Feature Request" etiketi ekleyin
3. Özelliği detaylı açıklayın
4. Kullanım senaryoları verin

## 🔧 Kod Katkısı

1. Fork yapın
2. Yeni branch oluşturun (`git checkout -b feature/amazing`)
3. Değişikliklerinizi commit edin
4. Branch'e push yapın
5. Pull Request açın

## 📝 Kod Standartları

- Minecraft fonksiyonları için `.mcfunction` uzantısı
- JSON dosyaları düzgün formatlanmış olmalı
- Açıklayıcı komutlar ekleyin

## 🧪 Test

Pull Request açmadan önce değişikliklerinizi test edin:
- `/reload` ile yükleyin
- Menüleri test edin
- Çok oyunculu test edin

## 📖 Dokümantasyon

Yeni özellikler ekliyorsanız:
- README.md'yi güncelleyin
- Wiki'ye örnek ekleyin
- Kullanım komutlarını belirtin
